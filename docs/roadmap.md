# Roadmap

## Fase 1 - Protótipo Navegável

- tela inicial;
- tela de cálculo;
- tela de resultado;
- onboarding inicial;
- renda mensal local;
- preferências locais;
- comparações tangíveis;
- testes automatizados do core e storage inicial;
- sem persistência complexa além do perfil inicial;

## Fase 2 - MVP Funcional

- histórico local;
- geração de card como imagem;
- compartilhamento nativo;
- ajustes de UI;
- testes de componentes React Native;
- testes de navegação;
- máscaras de entrada monetária;
- tela de configurações;
- SQLite local;
- analytics;
- crash reporting.

## Fase 3 - MVP Publicável

- polimento visual;
- revisão de textos;
- ícone e splash screen;
- termos de uso e privacidade.

## Fase 4 - Beta Fechado

- ajustes de UX;
- novos templates;
- correção de bugs;
- validação de frases;
- medição de share rate;
- testes com usuários reais.

## Fase 5 - Lançamento Público

- publicação nas lojas;
- página simples de divulgação;
- vídeos curtos de demonstração;
- acompanhamento de métricas.

## Fora do MVP

- login;
- sincronização;
- backend próprio;
- OCR;
- scraping;
- integração bancária;
- IA generativa;
- controle financeiro completo.
