# Analytics

## Ferramenta Recomendada

PostHog.

## Eventos Principais

- app_opened
- onboarding_started
- profile_created
- calculation_started
- calculation_completed
- share_card_generated
- share_clicked
- share_completed
- premium_screen_viewed
- premium_purchased
- history_opened
- reference_value_edited

## Métrica Central

```text
share_rate = cálculos compartilhados / cálculos concluídos
```

Meta inicial: `share_rate >= 15%`.

## Privacidade

Não enviar renda do usuário, nome detalhado da compra ou dados sensíveis para analytics.
