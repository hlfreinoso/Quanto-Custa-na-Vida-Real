# Core

Pacote para regras puras e reutilizaveis do produto.

Este pacote deve conter calculos, comparacoes, score, insights, formatacao e validacoes. Ele nao deve depender de React Native, Expo, storage local, billing, analytics ou APIs nativas.

## Estrutura Planejada

```text
src/
  calculations/
  comparisons/
  scoring/
  insights/
  formatting/
  validators/
```
